PRE-SESSION PROTOCOL (mandatory — run before every session)
═══════════════════════════════════════════════════════════════

Step 1 — FIELD_CLEARING_PROTOCOL
  Acknowledge and release prior session residue.
  "The trade I lost yesterday: I see it, I learn, I release the charge."
  Uncleaned field = prior emotion contaminates current signal.

Step 2 — CIRCLE_CASTING
  (a) Set aside: explicitly name external noise being left at the door.
  (b) Centre: state current structural reality in one sentence.
  (c) Consecrate: PRANAYAMA — box breathing (4-4-4-4 × 3 rounds).
  (d) Open: NKB_UNCONDITIONAL_ATTENTION — "I have no position on
      what this signal should say."

Step 3 — DAN_TIAN_CHECK
  Which center is dominant right now?
  Lower (fear/survival) → ARCHITECTURE_PAUSE before proceeding.
  Middle (emotion) → HIGH_AROUSAL protocol, reduce FIRE_MODE.
  Upper (clarity) → proceed to signal evaluation.

Step 4 — OPERATOR_STATE assessment
  SHUTDOWN / DISTORTED / UNDERACTIVE / OVERSTIMULATED /
  OPTIMAL / STHITAPRAJNA / HANUMAN
  If below OPTIMAL: no SNIPER entries until state shifts.

Step 5 — IDA_PINGALA_BALANCE check
  IDA excess (over-receptive, waiting too long)? → LOCK_MODE + deadline.
  PINGALA excess (over-active, forcing entries)? → DRIFT_MODE + breath.
  Balanced? → proceed.

SECTION 5 — OPERATOR STATE HIERARCHY (7 levels)
═══════════════════════════════════════════════════════════════

Level 1: SHUTDOWN — No execution.
Level 2: DISTORTED — Block new positions.
Level 3: UNDERACTIVE — LOCK_MODE + deadline.
Level 4: OVERSTIMULATED — Halve size, 2 confirmations.
Level 5: OPTIMAL — Normal pipeline.
Level 6: STHITAPRAJNA — OPTIMAL under extreme pressure.
         PROCESS_QUALITY stable regardless of P&L trajectory.
Level 7: HANUMAN — Zero ego, pure instrument.
         "Is any part of this decision about ME?"
         If YES: ego contamination. Reduce FIRE_MODE.
         If NO: HANUMAN_STATE. Full pipeline capacity.

GUNA_CLASSIFIER (unifies AROUSAL + SONIC + MATURATION):
  TAMAS: inertia, TAMASIC conditions → missed LEICESTER_PRIME.
  RAJAS: restless, RAJASIC conditions → CHAOS_ENTRY risk.
  SATTVA: clarity, SATTVIC conditions → SNIPER eligible.
  Only SATTVIC decisions qualify for SNIPER entries.

NISHKAMA_KARMA_GATE:
  Outcome attachment markers detected?
  (escalating commitment, averaging without thesis upgrade,
  holding through stop)
  → KARMA_CONTAMINATED. Reduce FIRE_MODE one level.

TAT_TVAM_ASI_DIVERGENCE:
  operator_arousal_state − market_arousal_state.
  Positive divergence = independent view = potential edge.
  Zero divergence = cognitive fusion with crowd = no edge.
  Negative divergence = operator more fearful than market = STOP.

HANUMAN_PRE_ENTRY CHECK:
  "Is any part of this decision about me rather than the structure?"
  If YES: wait. If NO: proceed.
