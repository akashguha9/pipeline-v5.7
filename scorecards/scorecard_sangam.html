SSSC State Machine — Four Relational States
CONFLUENCE
All flows aligned. D_total < 0.2. Three rivers flowing the same direction. High agreement but potentially already priced.
→ Check OAD. Low OAD = SHADOW_BOOK (too obvious). High OAD = SNIPER (rare consensus-wrong case).
PARALLEL NON-MIXING
Flows visible, directions different, no convergence. D_total [0.2–0.5], H < 0.3. Like Sangam where flows coexist.
→ WAIT_CONFIRMATION or PROBE. Thesis ambiguity. Reduce size if entering.
BOUNDARY TENSION
Two flows agree, one sharply disagrees. One D_pair > 0.6. The dissenting flow contains the Raman signal — investigate it.
→ Check OUTSIDER_ADVANTAGE_DETECTOR + TSL. The dissenter may be the edge signal. SNIPER if OAD high.
SUBSURFACE CONTROL
H > 0.6. Surface flows weak/noisy. VISOR flags high-volume move with no news catalyst. Saraswati is active.
→ Follow hidden current direction. LOW_SIZE_ENTRY due to opacity. Run TASTE_SIGNAL_LAYER check.
